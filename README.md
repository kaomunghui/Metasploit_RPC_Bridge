# Metasploit_RPC_Bridge

python透過RPC控制Metasploit

## Install Requirement
```
pip install pymetasploit3
```

## Download Metasploit
```
https://www.metasploit.com/
```


## Start Metasploit 與 RPC
```
執行 msfconsole.bat
```
```
load msgrpc Pass=password
```

## Example CMD:
```
python python_main.py 192.168.56.1 20-450
```
